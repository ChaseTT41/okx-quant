## Vision识图核心发现 (9大发现)

### 1. Low1/Low2入场计数系统
- Low1 = 上升趋势第一次回调低点 → "打底", 胜率较低
- Low2 = 上升趋势第二次回调低点 → "出击", 胜率显著更高
- High1/High2 = 下降趋势同理
- "有效信号K + 入场K + Low2" = 最高胜率组合

### 2. 信号K + 入场K 两步确认 (最核心)
- 信号K = 非趋势K线 (DOJI/PINBAR/INSIDE/小实体) → "方向犹豫"
- 入场K = 趋势K线 (TREND_BULL/TREND_BEAR) → "方向重新确认"
- 两步缺一不可，只看第1根顺趋势确认K

### 3. 不要在强动能K线追单
- 大实体趋势K后禁止立即入场
- body >= 2×avg + 远离关键位 → 追单警报
- 等回调到关键位 + 信号K确认

### 4. Pinbar战法2.0
- 必须关键位 + 趋势末端 同时满足
- 突破入场: 挂Pinbar高点上方
- 回调入场: 等回到Pinbar 50%
- 分组: 第一组/第二组集群信号更强

### 5. 趋势反转三步确认
- 上升→下降: ①破前低 → ②不创新高 → ③创新低
- 下降→上升: ①破前高 → ②不创新低 → ③创新高

### 6. 末端旗形/楔形三推 = 反转预警
- 末端旗形: 趋势末尾紧凑旗形, 能量耗尽
- 楔形三推: 三次推动幅度递减 → 即将反转

### 7. SB结构 (Second Breakout)
- 第1次突破(=假突破) → 收回 → 第2次突破(=真突破)
- 第2次胜率更高，与123法则、2B法则本质相通

### 8. 三步读图法
① 定级别(选对周期) → ② 画结构(识别骨架) → ③ 找信号(等待入场)

### 9. 入场三问
① 趋势对吗? ② 位置对吗? ③ 信号对吗?(信号K+入场K)

## 三大交易铁律
1. 顺大势逆小势
2. 第一次机会是陷阱
3. 只做看得懂的行情
